# Installation

- Clone LIBSVM branch
`git clone git@github.com:milanflach/LIBSVM.jl.git --branch mutating_versions`
- then in julia
`Pkg.build("LIBSVM")`
- Clone package
`Pkg.clone("https://github.com/milanflach/MultivariateAnomalies.jl")`
